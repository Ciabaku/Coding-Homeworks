Lista dei Tag Inline HTML

Nella tabella di seguito troverete tutti i Tag Inline HTML.
Di ciascuno viene fornita una piccola descrizione della funzione che svolge, un esempio e il relativo link per approfondire meglio l'argomento.

Definizione generale:

Un Tag Inline non viene eseguito su una nuova riga di testo e occupa solo la larghezza necessaria.


Tag Inline Descrizione Esempio Approfondisci
a Definisce un'ancora o un collegamento tra una pagina e l'altra. example.com
abbr Definisce un'abbreviazione. VVFF
acronym Definisce un'acronimo. ! WARNING: Non supporato in HTML5 !
address Definisce le informazioni di contatto dell'autore della pagina/articolo.
Scritto da: Luca Biagetti
Classe: 3BI
b Definisce un testo in grassetto, senza però assegnargli un'importanza extra. grassetto
bdo Definisce la direzione del testo. ciao
big Definisce un testo più grande. ! WARNING: Non supporato in HTML5 !
button Definisce un bottone clickabile.
cite Definisce il titolo di un'opera. La Divina Commedia di Dante Alighieri
code Definisce un testo "codice del computer". computer
dfn Definisce l'istanza di definizione di un termine. H20 è un composto di idrogeno e ossigeno.
em Definisce un testo enfatizzato. enfatizzato
i Definisce un testo corsivo. corsivo
img Definisce un immagine.
input Definisce un campo di input dove l'utente può inserire dei dati.
kbd Definisce un testo "keyboard input". keyboard
label Definisce un'etichetta, può avere diverse funzioni in base al tipo.
map Definisce un'immagine mappa clickabile dall'utente.
object Definisce un elemento embed. ex: video.
output Definisce la rappresentazione del risultato di un'operazione. = ...
q Definisce una piccola citazione. Ciao a tutti! Disse Luca sottovoce.
samp Definisce un testo "output programma". output programma
script Definisce uno script. script in esecuzione..
select Definisce un menù a tendina.
small Definisce un testo più piccolo. testo piccolo
span Definisce un gruppo di elementi inline. Luca ha gli occhi verdi
strong Definisce un testo importante. importante
sub Definisce un testo pedice. H2O
sup Definisce un testo apice. 103
textarea Definisce una casella di input di testo.
time Definisce una data/orario. Natale è il 25 Dicembre
tt Definisce un testo "teletype". ! WARNING: Non supporato in HTML5 !
var Definisce un testo "variable". variable
