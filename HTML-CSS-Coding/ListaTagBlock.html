Lista dei Tag Block HTML

Nella tabella di seguito troverete tutti i Tag Block HTML.
Di ciascuno viene fornita una piccola descrizione della funzione che svolge, un esempio e il relativo link per approfondire meglio l'argomento.

Definizione generale:

Un Tag Block inizia sempre su una nuova linea e occupa l'intera larghezza disponibile.
